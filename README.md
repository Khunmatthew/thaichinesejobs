# ThaiChineseJobs.com (MVP)

Thailand’s leading bilingual (Chinese ↔ Thai) recruitment platform.

## Status

This is the MVP version built with Next.js 15 + Supabase.

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

- Vercel ready
- Supabase backend

## Features
- Professional landing page
- Advanced job search & filters
- Multi-step job posting form
- Employer & Seeker dashboards
- AI matching architecture ready
